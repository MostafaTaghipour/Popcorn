## Important libraries
- react 16.9.0
- react-native 0.63.2
- expo 37.0.9
- react-navigation
- react-native-paper
- redux
- redux-thunk
- axios

## Start Expo

1.  run `yarn`
2.  run `yarn start`

## Run iOS or Android

run `yarn ios` or `yarn android`;

## Run test

run `yarn test`
